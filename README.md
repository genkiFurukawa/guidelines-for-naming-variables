# guidelines-for-naming-variables
変数名を決める時の指針。変数名を決める時に迷うことが多いので、調べたことはこのリポジトリにまとめる。

## CS分野特有の反対語
* get <=> set
* deep <=> shallow
* allocate <=> deallocate
* enqueue <=> dequeue
* head <=> tail
* expand <=> collapse
* inflate <=> deflate

## 止める
* stop
    - beginしたものを中止する時に使うイメージ？
* quit
    - quitの後に目的語がつけば、他動詞。
* exit
    - 作業場からでるイメージ。なので、その作業場自体は動作していることに留意。enterしたものが終わる時に使うイメージ？
* abort
    - 異常が発生した時に中断させるイメージ。
* cancel
    - ?
* shutDown
    - ?
* terminate
    - 終わらせる。beginしたものを終わらせる時に使うイメージ？
* kill
    - プロセスを終了する時に使う。
* break
    - 処理を中断する。forやwhileから抜け出す時にも使う動詞。enterしたものに対して、中断する時に使うイメージ？
